# Project Planning

Skills for turning ideas into executable, well-formed backlogs.

| Skill | What it does |
| --- | --- |
| [`plan-project`](./plan-project/SKILL.md) | User-invoked (`/plan-project`). Takes a project from "I have ideas" — plus any design docs or ADRs — to a review-ready backlog: reconciled intake, vertical-slice breakdown, mini-PRD tickets, dependency-aware sequencing, capacity-aware assignment, and shaping for Jira or Linear. Codebase-aware; produces markdown, never live writes. |
